# IT1214-Warehouse-Inventory-System

## Project Description
This is a Java-based Warehouse Management System developed for the IT1214 assignment. It uses Object-Oriented Programming (OOP) to manage inventory. The system allows users to add items, remove them, update stock quantities, and search for specific products by their ID.

## How to Run the Program
1. Open your command prompt or terminal.
2. Compile the files using:
   javac Warehouse.java
3. Run the program using:
   java Warehouse
